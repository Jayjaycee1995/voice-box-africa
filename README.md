## Getting Started

### Prerequisites

- Node.js 18+
- PHP 8.2+
- Composer
- PostgreSQL

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   # Frontend
   npm install
   
   # Backend
   cd laravel-backend
   composer install
   ```

3. Set up environment variables
4. Run migrations
5. Start development servers
